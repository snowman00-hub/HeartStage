# 🏗️ 1팀 주간 빌드 노트

## 📅 빌드 정보
- 빌드명: 주간 빌드1
- 빌드 날짜: 2025-11-14  
- Unity 버전: Unity 6.0 (6000.0.60f1)
- 플랫폼: Android  

---

## 🚀 구현된 기능
- 로비 UI 추가 (배치만, 기능X)
- 캐릭터 무대에 배치 후 시작 버튼 누를시 전투 시작
- 적들은 웨이브 형식으로 위에서 아래로 내려오며, 벽이 공격 사정거리안에 들어올시 공격한다.
- 캐릭터들은 적들이 사정거리 안에 들어올시 자동으로 공격을 발사한다.
- 몬스터는 피격시 자기 HP바 게이지가 0에서 1로 채워지고, 다 채워지면 사라진다.
- Wave의 모든 몬스터를 처치할 시 자동으로 다음 Wave로 넘어간다.

---

## 📝 임시로 처리한 부분
- 해당 스테이지의 웨이브를 모두 클리어시 승리 반응이 있는 대신 다음 스테이지의 웨이브를 계속해서 소환
- 에셋 : 모든 에셋은 테스트용
- 옵션 버튼 대신 로비로 가는 버튼으로 임시 처리

---

## 🔧 플레이하는 방법
- 로비 -> PlayButton Click Or 로비 -> 전투탭 -> 1-1 -> Play Button 누를시 스테이지 시작
